# 运行与环境配置指南

本文为基于 HarmonyOS 官方文档整理的主题导读，聚合运行、模拟器、真机与环境配置相关要点。

HarmonyOS 支持在真机、模拟器以及远程调试设备上运行应用。

## 1. 模拟器运行 (Emulator/Simulator)
模拟器是开发过程中最常用的运行环境。

### 本地模拟器 (Local Emulator)
*   **前提**：PC 支持虚拟化技术（VT-x/AMD-V）并已在 BIOS 中开启。
*   **启动**：在 DevEco Studio 中打开 `Tools > Device Manager`，选择 `Local` 选项卡，点击 `+ Create` 创建并启动。
*   **注意**：本地模拟器支持断网调试，响应速度较快。

### 远程模拟器 (Remote Simulator)
*   **前提**：网络畅通且已登录华为账号。
*   **启动**：在 `Device Manager` 中选择 `Remote` 选项卡，选择对应机型启动。

## 2. 真机运行 (Real Device)
真机调试支持硬件交互、性能表现及真实用户体验测试。

### 准备工作
*   **系统版本**：HarmonyOS NEXT Developer Beta1 或更高。
*   **开启调试**：在 `设置 > 关于手机` 连续点击“版本号” 7 次开启开发者模式。在 `系统 > 开发者选项` 中打开 “USB调试”。
*   **应用签名**：真机运行必须先完成[应用签名](应用签名指南.md)。

### 连接方式
*   **USB 连接**：使用数据线连接，并在手机弹窗中选择“始终允许”。
*   **无线调试**：
    1.  手机与 PC 处于同一 WLAN。
    2.  手机开发者选项中开启“无线调试”。
    3.  通过 `hdc tconn [IP]:[Port]` 手动连接，或在 DevEco Studio 运行设备列表中查找。

## 3. 常见问题与故障排查 (FAQs)
### 模拟器相关
*   **启动失败（提示 Virtualization/Hyper-V）**：
    *   确保 BIOS 中已开启 `VT-x` 或 `AMD-V`。
    *   Windows 用户若使用 HAXM，需关闭 `Hyper-V`；若使用系统自带加速，需开启 `Windows Hypervisor Platform`。
*   **黑屏或显示异常**：
    *   更新宿主机显卡驱动。
    *   在模拟器设置中尝试切换图形渲染模式（OpenGL/Software）。
*   **万能修复手段**：
    *   **Wipe Data**：在 Device Manager 中选择模拟器，点击 `Wipe Data` 清除缓存数据。
    *   **Cold Boot**：强制冷启动，不加载之前的快照状态。
*   **无法识别设备（端口冲突）**：
    *   检查 5037 等端口是否被其他手机助手占用。
    *   使用 `hdc kill` 和 `hdc start` 重启服务。

### 真机相关
*   **无法识别设备**：
    *   检查数据线或尝试更换接口。
    *   使用 `hdc list targets` 确认设备是否在线。
*   **Wearable 设备**：仅支持无线连接，不支持 USB 调试。
