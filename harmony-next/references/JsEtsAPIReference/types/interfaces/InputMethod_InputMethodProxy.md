# InputMethod_InputMethodProxy

```ets
typedef struct InputMethod_InputMethodProxy InputMethod_InputMethodProxy
```

**概述**

应用与输入法服务之间的交互代理对象，应用可通过此对象调用输入法服务的相关接口，并接收输入法服务的事件回调。

起始版本： 12

相关模块： [InputMethod](InputMethod.md)

所在头文件： [inputmethod_inputmethod_proxy_capi.h](inputmethod_inputmethod_proxy_capi.h.md)
