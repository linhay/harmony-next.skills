# ImageBitmap对象

  ![image](public_sys-resources/note_3.0-zh-cn.webp)

从API version 7开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。

ImageBitmap对象由[OffscreenCanvas对象](OffscreenCanvas对象.md)的[transferToImageBitmap()](OffscreenCanvas对象.md#ZH-CN_TOPIC_0000002553360891__transfertoimagebitmap)方法生成，存储了offscreen canvas渲染的像素数据。

**属性**

| 属性 | 类型 | 描述 |
| --- | --- | --- |
| width | number | ImageBitmap的像素宽度。 |
| height | number | ImageBitmap的像素高度。 |
