# Input_KeyState

```ets
typedef struct Input_KeyState Input_KeyState
```

#### 概述

定义按键信息，用于标识按键行为。例如，“Ctrl”按键信息包含键值和键类型。

**起始版本：** 12

**相关模块：**[input](input.md)

**所在头文件：**[oh_input_manager.h](oh_input_manager.h.md)

**相关接口：**

名称描述[OH_Input_CreateKeyState](oh_input_manager.h.md#ZH-CN_TOPIC_0000002529445539__oh_input_createkeystate)创建按键状态的枚举对象。[OH_Input_DestroyKeyState](oh_input_manager.h.md#ZH-CN_TOPIC_0000002529445539__oh_input_destroykeystate)销毁按键状态的枚举对象。
