# @ohos.application.NotificationSubscriber[ExtensionContext](../../topics/misc/ExtensionContext.md) (通知订阅扩展上下文)

NotificationSubscriber[ExtensionContext](../../topics/misc/ExtensionContext.md)模块提供通知订阅者扩展能力。


本模块首批接口从API version 22开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。

本模块接口仅可在Stage模型下使用。

#### 导入模块

```ets
import { NotificationSubscriberExtensionContext } from '@kit.NotificationKit';
```

#### NotificationSubscriber[ExtensionContext](../../topics/misc/ExtensionContext.md)

NotificationSubscriberExtensionContext继承自[ExtensionContext](ExtensionContext.md)类，是NotificationSubscriberExtensionAbility的上下文。

**系统能力**：SystemCapability.Notification.Notification
