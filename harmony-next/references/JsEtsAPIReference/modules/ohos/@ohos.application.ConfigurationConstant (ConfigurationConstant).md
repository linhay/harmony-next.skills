# @ohos.application.ConfigurationConstant (ConfigurationConstant)

ConfigurationConstant模块提供配置信息枚举值定义的能力。

本模块首批接口从API version 8开始支持，从API version 9废弃，替换模块为[@ohos.app.ability.ConfigurationConstant (ConfigurationConstant)](@ohos.app.ability.ConfigurationConstant (环境变量相关的常量定义).md)。后续版本的新增接口，采用上角标单独标记接口的起始版本。

#### 导入模块

```ets
import ConfigurationConstant from '@ohos.application.ConfigurationConstant';
```

#### ColorMode

表示颜色模式的枚举。

**系统能力**：SystemCapability.Ability.AbilityBase

名称值说明COLOR_MODE_NOT_SET-1未设置颜色模式。COLOR_MODE_DARK0深色模式。COLOR_MODE_LIGHT1浅色模式。
