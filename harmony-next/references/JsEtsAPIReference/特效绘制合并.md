# 特效绘制合并

用于对背景模糊等特效进行绘制合并。

从API version 12开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。

#### useEffect

useEffect(value: boolean): T

用于对背景模糊等特效进行绘制合并。

**元服务API：** 从API version 12开始，该接口支持在元服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**参数：**

参数名类型必填说明valueboolean是

控制组件是否继承EffectComponent组件的特效属性参数，从而合并绘制特效。

useEffect为true时子组件继承EffectComponent组件的特效属性参数，为false时子组件不继承EffectComponent组件的特效属性参数。

默认值：false

**返回值：**

类型说明T返回当前组件。

#### useEffect14+

useEffect(useEffect: boolean, effectType: EffectType): T

用于设置组件是否应用窗口定义的效果模板。

**元服务API：** 从API version 14开始，该接口支持在元服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**参数：**

参数名类型必填说明useEffectboolean是

控制组件是否应用窗口定义的效果模板。

useEffect为true时表示应用窗口定义的效果模板。

默认值：false

effectType[EffectType](特效绘制合并.md#ZH-CN_TOPIC_0000002529444805__effecttype14)是

设置组件应用窗口定义的效果模板。

默认值：EffectType.DEFAULT

**返回值：**

类型说明T返回当前组件。

#### useEffect18+

useEffect(useEffect: Optional<boolean>, effectType?: EffectType): T

用于设置组件是否应用窗口定义的效果模板。与[useEffect14+](#ZH-CN_TOPIC_0000002529444805__useeffect)相比，useEffect参数新增了对undefined类型的支持。

**元服务API：** 从API version 18开始，该接口支持在元服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**参数：**

参数名类型必填说明useEffectOptional<boolean>是

控制组件是否应用窗口定义的效果模板。

useEffect为true时表示应用窗口定义的效果模板。

默认值：false

当useEffect的值为undefined时，维持之前取值。

effectType[EffectType](特效绘制合并.md#ZH-CN_TOPIC_0000002529444805__effecttype14)否

设置组件应用窗口定义的效果模板。

默认值：EffectType.DEFAULT

**返回值：**

类型说明T返回当前组件。
