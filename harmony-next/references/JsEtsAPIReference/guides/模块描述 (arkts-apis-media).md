# 模块描述

本模块首批接口从API version 6开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。

媒体子系统为开发者提供一套简单且易于理解的接口，使得开发者能够方便接入系统并使用系统的媒体资源。

媒体子系统包含了音视频相关媒体业务，提供以下常用功能：

-

音视频播放（[AVPlayer](../types/interfaces/Interface (AVPlayer).md)9+）

-

音视频录制（[AVRecorder](../types/interfaces/Interface (AVRecorder).md)9+）

-

视频转码（[AVTranscoder](../types/interfaces/Interface (AVTranscoder).md)12+）

-

获取音视频元数据（[AVMetadataExtractor](../types/interfaces/Interface (AVMetadataExtractor).md)11+）

-

获取视频缩略图（[AVImageGenerator](../types/interfaces/Interface (AVImageGenerator).md)12+）

-

屏幕录制（[AVScreenCaptureRecorder](../types/interfaces/Interface (AVScreenCaptureRecorder).md)12+）
