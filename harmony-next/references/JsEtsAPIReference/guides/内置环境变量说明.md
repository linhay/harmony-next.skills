# 内置环境变量说明

本模块首批接口从API version 7开始支持，后续版本的新增接口，采用上角标单独标记接口的起始版本。

#### ColorMode

系统当前深浅色模式。

**元服务API：** 从API version 11开始，该接口支持在元服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

名称值说明LIGHT0浅色模式。DARK1深色模式。

#### LayoutDirection

系统的布局方向类型。

**元服务API：** 从API version 11开始，该接口支持在元服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

名称值说明LTR0从左向右布局。RTL1从右向左布局。Auto8+2自动布局，跟随系统。
