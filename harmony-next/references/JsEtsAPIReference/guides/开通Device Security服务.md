# 开通Device Security服务

本文根据华为开发者官网 `devicesecurity-deviceverify-activateservice` 页面整理为离线版入口。
来源：<https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/devicesecurity-deviceverify-activateservice>
更新时间：2026-04-20 06:34:33

在开通 Device Security 服务前，需要先完成应用开发基础准备，然后再到 AppGallery Connect 中启用对应能力。

该页覆盖的核心流程：
- 登录 `AppGallery Connect`，进入目标项目。
- 打开“开放能力管理”页签。
- 按需开通具体能力，例如应用设备状态检测、安全检测、可信应用服务、业务风险检测、数字盾服务等。

需要注意的约束：
- “可信应用服务”等部分能力需要先申请进入允许清单。
- “涉诈剧本检测”“数字盾服务”等能力通常要求先完成企业开发者实名认证，并提交申请材料等待审核。
- 某些能力开通后还需要继续申请调试或发布使用所需的 `Profile` 文件。
