# ArkUI_HostWindowInfo

```ets
typedef struct ArkUI_HostWindowInfo ArkUI_HostWindowInfo
```

#### 概述

定义窗口属性的HostWindowInfo类信息。

**起始版本：** 15

**相关模块：**[ArkUI_NativeModule](../system-services/ArkUI_NativeModule.md)

**所在头文件：**[native_type.h](../../capi/headers/native_type.h.md)

**相关接口：**

名称描述[OH_ArkUI_NodeUtils_GetWindowInfo](../../capi/headers/native_node.h.md#ZH-CN_TOPIC_0000002529285071__oh_arkui_nodeutils_getwindowinfo)获取节点所属的窗口信息。[OH_ArkUI_HostWindowInfo_GetName](../../capi/headers/native_type.h.md#ZH-CN_TOPIC_0000002497445102__oh_arkui_hostwindowinfo_getname)获取HostWindowInfo对象中的窗口名称。[OH_ArkUI_HostWindowInfo_Destroy](../../capi/headers/native_type.h.md#ZH-CN_TOPIC_0000002497445102__oh_arkui_hostwindowinfo_destroy)销毁HostWindowInfo对象。
