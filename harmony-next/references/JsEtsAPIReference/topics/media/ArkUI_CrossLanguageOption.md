# ArkUI_CrossLanguageOption

```ets
typedef struct ArkUI_CrossLanguageOption ArkUI_CrossLanguageOption
```

#### 概述

定义跨语言配置项。

**起始版本：** 15

**相关模块：**[ArkUI_NativeModule](../system-services/ArkUI_NativeModule.md)

**所在头文件：**[native_type.h](../../capi/headers/native_type.h.md)

**相关接口：**

名称描述[OH_ArkUI_NodeUtils_SetCrossLanguageOption](../../capi/headers/native_node.h.md#ZH-CN_TOPIC_0000002529285071__oh_arkui_nodeutils_setcrosslanguageoption)设置目标节点跨语言设置属性的能力。[OH_ArkUI_NodeUtils_GetCrossLanguageOption](../../capi/headers/native_node.h.md#ZH-CN_TOPIC_0000002529285071__oh_arkui_nodeutils_getcrosslanguageoption)获取目标节点跨语言设置属性的配置项。[OH_ArkUI_CrossLanguageOption_Create](../../capi/headers/native_type.h.md#ZH-CN_TOPIC_0000002497445102__oh_arkui_crosslanguageoption_create)创建跨语言配置项实例。[OH_ArkUI_CrossLanguageOption_Destroy](../../capi/headers/native_type.h.md#ZH-CN_TOPIC_0000002497445102__oh_arkui_crosslanguageoption_destroy)销毁跨语言配置项实例。[OH_ArkUI_CrossLanguageOption_SetAttributeSettingStatus](../../capi/headers/native_type.h.md#ZH-CN_TOPIC_0000002497445102__oh_arkui_crosslanguageoption_setattributesettingstatus)设置配置项中是否允许跨语言修改属性。[OH_ArkUI_CrossLanguageOption_GetAttributeSettingStatus](../../capi/headers/native_type.h.md#ZH-CN_TOPIC_0000002497445102__oh_arkui_crosslanguageoption_getattributesettingstatus)获取配置项中是否允许跨语言修改属性。
