# Input_CustomCursor

```ets
typedef struct Input_CustomCursor Input_CustomCursor
```

#### 概述

自定义鼠标光标像素图资源。

**起始版本：** 22

**相关模块：**[input](input.md)

**所在头文件：**[oh_input_manager.h](../../capi/headers/oh_input_manager.h.md)

**相关接口：**

名称描述[OH_Input_CustomCursor_Create](../../capi/headers/oh_input_manager.h.md#ZH-CN_TOPIC_0000002529445539__oh_input_customcursor_create)创建自定义鼠标光标资源对象。[OH_Input_CustomCursor_Destroy](../../capi/headers/oh_input_manager.h.md#ZH-CN_TOPIC_0000002529445539__oh_input_customcursor_destroy)销毁自定义鼠标光标资源对象。
