# Netstack

#### 概述

提供SSL/TLS证书链校验模块的C接口。

**起始版本：** 11

#### 文件汇总

| 名称 | 描述 |
| --- | --- |
| [net_ssl_c.h](net_ssl_c.h.md) | 定义SSL/TLS证书链校验模块C接口数据结构。 |
| [net_ssl_c_type.h](net_ssl_c_type.h.md) | 定义SSL/TLS证书链校验模块的C接口需要的数据结构。 |
| [net_websocket.h](net_websocket.h.md) | 定义websocket客户端模块的接口。 |
| [net_websocket_type.h](net_websocket_type.h.md) | 定义websocket客户端模块的C接口需要的数据结构。 |
| [net_http.h](net_http.h.md) | 定义HTTP请求模块的接口。 |
| [net_http_type.h](net_http_type.h.md) | 定义HTTP请求模块的C接口需要的数据结构。 |
