# NetworkBoost_MultiPathRecommendation

**概述**

多网推荐信息，用于注册多网推荐变化事件回调后，系统多网推荐状态发生变化的事件通知。

起始版本： 6.0.2(22)

相关模块： [NetworkBoost](NetworkBoost.md)

所在头文件： [network_boost_handover.h](network_boost_handover.h.md)

**汇总**

**成员变量**

| 名称 | 描述 |
| --- | --- |
| NetworkBoost_MultiPathAction action | 多网推荐动作。 |

**结构体成员变量说明**

**action**

```ets
NetworkBoost_MultiPathAction NetworkBoost_MultiPathRecommendation::action
```
