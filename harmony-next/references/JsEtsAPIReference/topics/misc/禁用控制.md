# 禁用控制

组件可交互状态下响应[点击事件](点击事件.md)、[触摸事件](触摸事件.md)、[拖拽事件](拖拽事件.md)、[按键事件](按键事件.md)、[焦点事件](焦点事件.md)、[鼠标事件](鼠标事件.md)、[轴事件](轴事件.md)、[悬浮事件](悬浮事件.md)、[无障碍悬浮事件](无障碍悬浮事件.md)、[手势事件](绑定手势方法.md)、[焦点轴事件](焦点轴事件.md)和[表冠事件](表冠事件.md)。

 从API version 7开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。

 禁用控制属性仅在按下时生效，交互过程中更改enabled属性无效。

#### enabled

enabled(value: boolean): T

设置组件是否可交互。当未设置enabled时，组件默认可交互。

**卡片能力：** 从API version 9开始，该接口支持在ArkTS卡片中使用。

**元服务API：** 从API version 11开始，该接口支持在元服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**参数：**

参数名类型必填说明valueboolean是

值为true表示组件可交互，响应点击等操作。

值为false表示组件不可交互，不响应点击等操作。

**返回值：**

类型说明T返回当前组件。

#### 示例

该示例通过enabled设置按钮可交互性。

```ets
// xxx.ets
@Entry
@Component
struct EnabledExample {

  build() {
    Flex({ justifyContent: FlexAlign.SpaceAround }) {
      // 点击时无响应
      Button('disable').enabled(false).backgroundColor(0x317aff).opacity(0.4)
      Button('enable').backgroundColor(0x317aff)
    }
    .width('100%')
    .padding({ top: 5 })
  }
}
```
