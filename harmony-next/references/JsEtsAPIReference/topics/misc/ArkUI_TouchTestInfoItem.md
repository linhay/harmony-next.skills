# ArkUI_TouchTestInfoItem

```ets
typedef struct ArkUI_TouchTestInfoItem ArkUI_TouchTestInfoItem
```

**概述**

定义触摸测试信息项。触摸测试信息项包含触摸测试中子组件的信息，可以通过[OH_ArkUI_TouchTestInfoItem_GetXXX](ui_input_event.h.md#ZH-CN_TOPIC_0000002522241032__oh_arkui_touchtestinfoitem_getx)系列接口获取子组件信息。

起始版本： 22

相关模块： [ArkUI_EventModule](ArkUI_EventModule.md)

所在头文件： [ui_input_event.h](ui_input_event.h.md)
